# Glossar

Kurzuebersicht zentraler Begriffe im Projekt.

- Bezirk (District): Oberster fachlicher Mandant.
- Gemeinde (Congregation): Untereinheit eines Bezirks.
- Gemeindegruppe (Congregation Group): Optionale Gruppierung mehrerer Gemeinden.
- Event: Terminobjekt im aktuellen Modell.
- ServiceAssignment: Dienstzuweisung (z. B. Gottesdienstleitung) zu einem Termin.
- PlanningSeries: Geplante Serie wiederkehrender Planungsslots (Zielmodell).
- PlanningSlot: Normative Planungsposition in der Matrix (Zielmodell).
- EventInstance: Tatsaechliche Ausfuehrung eines Planungsslots (Zielmodell).
- CalendarIntegration: Anbindung an externen Kalender (ICS, CalDAV, ...).
- OIDC: OpenID Connect fuer Login und Token-basierte Identitaet.
- PKCE: Erweiterung fuer sichere Authorization-Code-Flows in SPAs.
- RBAC: Role-Based Access Control.
- Membership: Rollen-/Scope-Zuordnung eines Benutzers.
- PUBLIC/INTERNAL Token: Export-Tokentypen fuer unterschiedliche Datensichtbarkeit.
